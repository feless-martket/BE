## 1. 📃 프로젝트 소개
- 프로젝트 이름: FEless-Market(간편한 온라인 식료품 쇼핑 플랫폼)
- 소개: 이 프로젝트는 사용자에게 신선한 식료품과 생활용품을 간편하게 구매할 수 있는 경험을 제공하는 온라인 쇼핑 플랫폼입니다. 사용자 친화적인 인터페이스와 효율적인 장바구니 및 결제 시스템을 통해 마켓컬리와 유사한 기능을 구현합니다.
